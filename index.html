What's New in v1.6.4

Project Dropdown Bug Fix

Fixed an issue where the active project's name would blend into the background highlight and become invisible when hovering over it in the Project Selection menu.

Menu Cleanup

Removed the "Install App (PWA)" button from the File menu to streamline the UI based on user feedback.

Update History

v1.6.3

Format on Save: Added a toggle in the Edit menu to automatically format code on Ctrl+S.

Word Wrap Toggle: Instantly toggle word wrapping right from the editor toolbar.

Duplicate File: Hover over files or assets in the sidebar to quickly clone them.

Premium Dropdowns: Replaced the default browser project selector with a sleek, animated dropdown.

v1.6.2

Critical UI Restoration: Fixed a massive code-truncation bug that broke the core interface. File creation, project selection, and file uploading are fully functional again.

v1.6.1

Upload UI Enhancements: The file upload area has been redesigned to be a prominent, clickable button zone with clear icons.

v1.6.0

Global Search & Replace: Search and replace code across every file in your project instantly (Ctrl+Shift+F).

Drag & Drop Files: Click and drag files in the sidebar to organize them into folders visually.

Local Snapshots: Back up your work locally with a Time Machine-like interface under the Project menu.

Asset Viewer & Minimap: View images/videos directly in editor tabs, added a CSS Color Picker, and a Code Minimap.

v1.5.3

Clean Slate Defaults: New projects now start with an empty index.txt.

In-App Warnings: Replaced all browser confirm alerts for Factory Reset with beautiful native modals.

v1.5.2

Project Reset Optimization: Restore to Default safely resets ONLY the current active project.

Factory Reset: Added a complete wipe option with double-confirmation.

JS Console Fix: Developer Console properly intercepts live code logs.

v1.5.1

Bug Fixes: Restored the premium styling for the Change File Type modal.

Infinite History: The update history tab now natively supports scrolling through all past updates infinitely.

v1.5.0

Split-Screen & Resizing: View two files side-by-side and drag borders to resize the workspace panels.

Path-Based Folders: Auto-organize files into collapsible folders by naming them with slashes.

JS Console: Intercept and log JavaScript errors in real-time beneath the preview.

Power Features: Added Prettier code auto-formatting and Quick Open (Ctrl+P).

v1.4.1

Nexus Convert Pro: Added integrated file converter links to the File menu.

v1.4.0

Project Manager: Centralized menu to manage and delete your existing local projects.

Factory Reset: Added a "Restore to Default" option to cleanly wipe all local data.

Asset Downloading: Introduced a download button for uploaded assets.

v1.3.3

Bulletproof Importer: Re-engineered the import logic to safely stream huge 1GB+ files without crashing V8 memory limits.

v1.3.0

File Extensions: Custom file extension builder with color-coded editor themes based on language.

Smooth UI: Completely rebuilt the UI with glass-morphism blurs and buttery-smooth CSS transitions on all interactive elements.
